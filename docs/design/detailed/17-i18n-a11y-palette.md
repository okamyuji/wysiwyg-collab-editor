# 17 i18n・アクセシビリティ・カラーパレット

## スコープ

UI言語切替、WCAG 2.1 AA準拠、16色固定カラーパレットのHEX値とコントラスト比、reduced-motion対応、文字色・背景色の異組合せ許容ルールを集約する。

## 契約

### UI言語

| 項目                 | 値                                                          |
| -------------------- | ----------------------------------------------------------- |
| 対応言語             | 日本語、英語(初期値はブラウザ `Accept-Language` 自動判定)   |
| 切替UI               | 設定画面で明示切替                                          |
| 永続化               | `users.locale`(`ja`/`en`)、ログイン時に上書き               |
| 翻訳ライブラリ       | `i18next`(クライアント側)、`i18next-fs-backend`(サーバー側) |
| 文字エンコーディング | UTF-8(NFC正規化)                                            |

### 異組合せの可否ルール

役割番号の異なる文字色と背景色の組合せ(計16×16=256通り中、推奨16通りを除く240通り)はビルド時にコントラスト比を算出する。

```text
contrast_ratio = (lum_lighter + 0.05) / (lum_darker + 0.05)
WCAG 2.1 AA基準: contrast_ratio >= 4.5(通常テキスト)
```

4.5未満となる組合せはUI上で選択不能化する。本MVPの初期見積もりとして、推奨16通り+淡色背景8種類×濃色文字8種類の有効組合せ約64通り=合計約80通り以上を確保する。

ビルド成果物 `dist/palette-allowed-combinations.json` を生成し、UIランタイムが読み込む。

### CSP `style-src-attr` ハッシュ算出との連携

許可される全組合せについて以下のCSSプロパティを文字列化し、SHA-256ハッシュを[13 CSPとサニタイズ](./13-csp-and-sanitization.md)の `style-src-attr` allowlistへ登録する。

```text
color: <fg_hex>; background-color: <bg_hex>;
```

ビルドパイプラインの順序: パレット組合せ算出 → CSPハッシュ算出 → 両出力JSONをサーバーが起動時に読込み。

### reduced-motion対応

| 検出     | `@media (prefers-reduced-motion: reduce)`                        |
| -------- | ---------------------------------------------------------------- |
| 適用範囲 | UIアニメーション全般(画面遷移、提案ハイライト、カーソルフェード) |
| 動作     | アニメーション無効化、即時状態遷移                               |

### キーボード操作

| 要件                         | 値                                            |
| ---------------------------- | --------------------------------------------- |
| 全機能のキーボード到達可能性 | 必達                                          |
| Tab順序の論理性              | 必達                                          |
| skip linkの提供              | 必達(エディタ本文へのジャンプ)                |
| フォーカスリングの可視性     | 2px、`outline-color: #1565c0`(役割11リンク色) |

### スクリーンリーダー対応

| 要件       | 値                                                                               |
| ---------- | -------------------------------------------------------------------------------- |
| ARIA roles | エディタは `role="textbox"`、コメントは `role="comment"`、提案は `role="status"` |
| aria-live  | 提案受諾/却下通知は `aria-live="polite"`                                         |
| ラベル     | 全フォームコントロールに `<label>` または `aria-label`                           |

### エラーコード

| コード     | HTTP | 意味                              |
| ---------- | ---- | --------------------------------- |
| `A11Y-001` | 400  | 16色パレット外HEX指定             |
| `A11Y-002` | 400  | コントラスト比4.5未満の組合せ指定 |

詳細は[19エラーコード登録簿](./19-error-code-registry.md)参照。
